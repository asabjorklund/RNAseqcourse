Isoform detection using RNA seq de novo Assembly¶

We are going to use two of the open source cutting edge RNA de novo assemblers during this practical. They are called Oases and Trinity. Independent assesment of de novo assembly programs showed that oases has a bit higher sensitivity at the cost of a bit worse specificty (ADD REFERENCE MAYBE FIGURE??).

cd $YOURWORKDIR mkdir deNovo cd deNovo # Soft link to all the reads ln -s /bubo/proj/g2013048/private/RNAseqWorkshop/deNovo/reads # Soft link to all the extra programs that will be used ln -s /bubo/proj/g2013048/private/RNAseqWorkshop/bin

Now that you are in the right folder and have created a soft link to the reads we are going to assemble we will use a script that will generate a sbatch script for each of the assemblers on the reads. The script will identify all the subfolders of your selected inFolder and create a assembly on all fastq files that are located in the same folder. In total we will do 24 different assemblies in this course

# load java module on uppmax module load java java -jar bin/HTStools.jar -p sbatch -deNovo -trinity -oases -pNr YOURPNR

/-email YOUR@EMAIL -i reads

To start all the sbatch scripts generated go to the automated created scripts- folder scripts and identify the newly created shell script file. Should be named [DATE]_PE_deNovo.sh

# go to the scripts folder cd scripts # Start the shell script that will start all the sbatch scripts sh [DATE]_PE_deNovo.sh

To see the actual calls for the two different assemblers you can go to reads_trinity/scripts and reads_oases/scripts respectively. They should be named [DATE]_0_trinity.sbatch and [DATE]_0_oases.sbatch.

cd $YOURWORKDIR cd deNovo/reads_trinity/scripts ls -lrt

The longest time for both programs should be under 5 minutes but now is a good time for a cup og java.
